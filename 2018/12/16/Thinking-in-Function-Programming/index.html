In Java

Java在8之前不支持函数式编程。Java8新增Stream API支持以一种声明的方式处理数据；
例如：

// java11
var list = List.of("a", "b", "c", "1");
var listUpper = list.parallelStream()
                    .filter(s -> !s.isEmpty())
                    .map(s -> s.toUpperCase())
                    .collect(Collectors.toList());
var sortedList = list.stream()
                    .sorted()
                    .collect(Collectors.joining(","));
